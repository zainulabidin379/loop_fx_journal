# LOOP

**LOOP** is a fully offline Flutter trading journal for forex and gold (XAUUSD) traders. Log trades, track performance, manage strategies, and review emotional patterns — all stored locally on your device with no backend or internet dependency.

## Features

- **Trade logging** — Instruments (XAUUSD, EURUSD, GBPUSD, and more), long/short, entry/SL/TP, lot size, notes, tags, emotions, and screenshot attachments
- **Close trade options** — Close at Take Profit, Stop Loss, or a custom exit price
- **Position calculator** — Suggested lot size from account balance, risk %, and stop distance
- **Dashboard** — Total P&L, win rate, avg R:R, profit factor, equity curve, win/loss breakdown, and performance by instrument, strategy, and emotion
- **Open trades** — Quick view of active positions from the dashboard
- **Streak tracker** — Current and longest win/loss streaks
- **Strategies** — Create, edit, and archive strategies with per-strategy performance stats
- **Weekly / monthly recap** — Auto-generated performance summaries
- **Biometric lock** — Optional fingerprint / Face ID gate on app open
- **Export / import** — JSON (full backup) and CSV (trades only) for offline data safety
- **Trade log reminders** — Optional local notification if no trade logged in 24 hours
